Joel Reznick's ASP Tetris Solver
-----------
To run the program, you need the following:

clingo 5.3.0
Python 3.7.0
Numpy
-----------
To run the program, open your command prompt and change the current
directory to wherever the ASP Project folder is located.

Then execute the gui.py script using your Python interpreter.

Ex for windows:
cd "(...)/ASP Project"
python gui.py
